For devices


Use below classes to set layout based on devices.

np-sm-col-* for small devices.
np-md-col-* for medium devices.
np-lg-col-* for large devices.
np-xl-col-* for extra large devices
Where * can be number from 1 to 12.

Alignment


Use below classes to set content alignment.

align-items-*
Where * can be start, end, center, baseline, stretch
align-content-*
Where * can be start, end, center, baseline, stretch
align-self-*
Where * can be start, end, center, baseline, stretch
justify-content-*
Where * can be start, end, center, between, around
